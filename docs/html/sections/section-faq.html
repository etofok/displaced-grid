Download Autohotkey v1.1 (not v2) here: https://www.autohotkey.com/ Download Displaced Grid Extract downloaded .zip Find "_Displaced GRID for Warcraft III.ahk" Open "_Displaced GRID for Warcraft III.ahk" with Autohotkey v1.1 Displaced Grid will appear in your Tray menu Displaced Grid is Off Displaced Grid is On Default Toggle — ScrollLock. Default Restart — CTRL + 3. Default Settings — CTRL + 6.
Displaced Grid overlays its set of keys on top of the in-game GRID keys: You press one button, but the game receives another. You must switch to GRID in game options. Your in-game layout must be set to GRID. This icon indicates whether hotkeys are currently remapped: Displaced Grid is Off Displaced Grid is On When Warcraft III window is not active (tabbed out) Displaced Grid has no effect. You can toggle Displaced Grid On or Off via a hotkey. Default toggle — ScrollLock. You can also double-click the icon in tray. p.s. Please note, when the layout is active you will struggle to type in chats because your keys are remapped. Displaced Grid is NOT a part of Warcraft III, technically - it doesn't even interact with the game. So it does not know that you want to type in chat.
You must use 'GRID' as your input keys to have ability quickcasts in Warcraft III. (this feature has not been fully developed by Blizzard, but it technically exists) Step 1. Select 'GRID' as your input keys. Step 2. Enable ability quickcasts in Settings in two clicks: Step 3. Locate War3Preferences.txt in User/Documents/Warcraft III Step 4. Write "Quickcast=1" You may choose to do it by hand: Step 2. Go to your %USERNAME%/Documents/Warcraft III folder Step 3. Open War3Preferences.txt Step 4. Change every QuickCast=0 to QuickCast=1 (CTRL + H) Step 5. Enjoy!

You can enable or disable modules in Settings.

Disabled modules will not be loaded and will not appear in the tray menu.
